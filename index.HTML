El primer documento que debes formar es index.html

Al abrir el archivo, la primera etiqueta que hay que agregar es doctype

Como aplicar CSS

Con CSS puedes mejorar la apariencia de tu pagina web, poner colores, estilos de letra, personalizar tu citio en general.

Algunas etiquetas pueden ser:

background para poner el color del panel

font-family: para el estilo de letra

margin: para poner margenes superiores (top) , inferiores (bottom)

Border: para bordes

Padding: para ajustar tu texto o listas dentro de un panel

Text-aline: para alinear un texto

width y height: para ajustar el ancho y alto de un panel

float: para hacer flotar nuestro elemento en algun lugar de nuestra pagina
